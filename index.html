Standards & Protocols
Attestation Standard
TRACE
Transparent Runtime Attestation for Confidential Execution. Hardware-rooted cryptographic receipts for every agent action — signed by the TEE, verifiable by anyone.
trace.agentrust-io.com
Live
Identity Standard
Agent Manifest
A structured declaration of an agent's capabilities, permissions, and data access policies. Machine-readable identity that operators and orchestrators can verify before invocation.
manifest.agentrust-io.com
Live
Protocol Extension
Confidential MCP
An extension to the Model Context Protocol that runs tool calls inside a Trusted Execution Environment. Prevents the host process from observing inputs, outputs, or intermediate state.
cmcp.agentrust-io.com
Live
